# Интерпретатор для языка программирования C

Учебный интерпретатор для ограниченного подмножества языка C, реализованный на C++. Программа выполняет полный цикл обработки исходного кода: лексический анализ, синтаксический анализ, генерацию промежуточного кода в виде польской записи и стековую интерпретацию.

## Возможности

Поддерживается следующее подмножество языка C:

- **Типы данных**: только целые числа (`int`).
- **Переменные**: простые целочисленные переменные (неявное объявление со значением 0 при чтении).
- **Массивы**: одномерные массивы фиксированного размера (объявление `int имя[размер];`, заполнение нулями).
- **Арифметические операции**: `+`, `-`, `*`, `/`, `%` (с проверкой деления на ноль).
- **Операции сравнения**: `<`, `>`, `<=`, `>=`, `==`, `!=` (только внутри условия цикла).
- **Присваивание**: `переменная = выражение;` или `массив[индекс] = выражение;`.
- **Цикл с постусловием**: `do { ... } while (условие);`.
- **Ввод/вывод**: `scanf(переменная);` и `printf(выражение);` (без форматных строк, по одному аргументу).

**Ограничения**: вещественные числа, строки, многомерные массивы, функции, сложные форматы `scanf`/`printf` не поддерживаются.

## Технические детали

| Параметр | Значение |
|----------|----------|
| Тип промежуточного кода | Польская запись (постфиксная) |
| Тип транслятора | Интерпретатор |
| Метод синтаксического разбора | Рекурсивный спуск |
| Язык реализации | C++ |
| Сборка | CMake или ручная компиляция |

## Сборка и запуск

### Требования
- Компилятор с поддержкой C++17 (g++, clang++, MSVC)
- Система сборки не требуется (можно компилировать вручную)

### Компиляция (пример для g++)

```bash
```bash
    g++ -std=c++17 -o interpreter lexer.cpp parser.cpp rpn.cpp interpreter.cpp main.cpp
